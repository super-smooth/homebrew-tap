# super-smooth's Homebrew Tap

This is the official Homebrew tap for CLI tools by super-smooth.

## Available Formulas

- **better-fg** - A more userfriendly and interactive version of fg

## Installation

```bash
# Install better-fg
brew install super-smooth/tap/better-fg

# Or install any other available formula
brew install super-smooth/tap/<formula-name>
```

## Usage

### better-fg

After installation, add this to your shell config:

```bash
eval "$(better-fg init)"
```

Then use the `bfg` command to interactively select background jobs.
